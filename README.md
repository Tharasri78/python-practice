# Python Practice Programs

This repository contains a collection of small Python programs created while learning core programming concepts.

These exercises focus on improving problem-solving skills and understanding basic Python logic such as conditionals, loops, and randomization.

## Programs Included

- Guess Game – number guessing game
- Password Generator – generates random passwords
- Quiz Program – simple question-based quiz
- Dice Roll Simulator – simulates rolling a dice
- Rock Paper Scissors – command line game
- Story Generator – random story generator

## Technologies Used

- Python 3

## Repository Structure

guess.py        – number guessing game  
passwords.py    – password generator  
quizz.py        – quiz application  
rolldice.py     – dice roll simulator  
rps.py          – rock paper scissors game  
storywords.py   – story generator  

## Purpose

This repository was created as part of self-learning and practice while studying Python programming fundamentals.
